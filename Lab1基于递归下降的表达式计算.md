# 基于递归下降的表达式计算

## 1. 实验目的

1. 深入理解词法分析：掌握如何将源代码字符串拆解为具有语义特征的 Token 序列。
2. 掌握递归下降分析法：理解自顶向下语法分析的工作原理，以及文法规则如何直接映射为程序函数。
3. 处理运算优先级：通过文法的层次结构解决加减乘除的结合性与优先级问题。

## 2. 实验内容

实现一个能够处理基础数学运算的解释器。

1. 基础要求：支持 `+`、`-`、`*`、`/` 及多层括号 `()`。
2. 进阶要求（可选）：支持变量赋值（如 `a = 10`）和变量参与运算。
3. 输入示例：`3 * (4 + 2) - 10 / 2`
4. 输出示例：`13.0`

## 3. 构造合适的文法

请对比下面三种文法的异同，从中选择一个，并基于它实现代码。

```text
E → T + T | T - T
T → F * F | F / F
F → NUM | (E)
```

以及

```text
E  → T E'
E' → + T E' | - T E' | ϵ
T  → F T'
T' → * F T' | / F T' | ϵ
F  → NUM | (E)
```

以及

```text
E → T + E | T - E
T → F * T | F / T | ϵ
F → NUM | (E)
```
